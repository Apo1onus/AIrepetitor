# 🎓 AI-Репетитор MVP | Демо-прототип интерфейса

Кликабельный прототип основного пользовательского сценария: `Диагностика → Карта пробелов → AI-разбор ошибки`.

🔗 **Live Demo**: [https://apo1onus.github.io/AIrepetitor/](https://apo1onus.github.io/AIrepetitor/)

## 📱 Что реализовано в демо
- Адаптивный мобильный интерфейс (iOS/Android)
- Имитация входной диагностики (2 вопроса)
- Цветовая карта тем с прогресс-барами
- Экран AI-разбора ошибки (структура ответа, алгоритм, закрепление)
- Переключение экранов без перезагрузки (SPA-логика)

## ⚙️ Технологический стек
- **Frontend**: Чистый HTML5 / CSS3 / Vanilla JS
- **Деплой**: GitHub Pages (статичный хостинг)
- **Планируемый бэкенд MVP**: Firebase (Auth, Firestore) + YandexGPT / GPT-4o mini + RAG-база ФИПИ

## 🚀 Как запустить локально
1. Клонируйте репозиторий: `git clone <url>`
2. Откройте `index.html` в любом браузере двойным кликом
3. Для просмотра в мобильном режиме: `Ctrl+Shift+M` (Chrome DevTools)
